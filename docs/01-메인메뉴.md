# 메인 메뉴

> 상위 문서: [`../PRD.md`](../PRD.md) §5.1

## 개요

콘솔 애플리케이션 실행 시 최초로 보여지는 화면. 하위 기능을 선택할 수 있는 진입점이며, 전체 시료에 대한 요약 정보를 함께 보여준다.

## 화면 구성

1. 전체 시료 요약 정보 (상단)
2. 메뉴 목록 (하단, 번호 선택 방식)

## 요약 정보 항목

메인 메뉴 진입/갱신 시 아래 정보를 표시한다.

- 등록된 전체 시료 종류 수
- 시료별 현재 재고 수량 요약 (또는 재고 부족/고갈 시료 개수 등 한눈에 파악 가능한 요약치)

> 표시 항목의 정확한 레이아웃은 자유롭게 결정한다. 다만 "담당자가 시스템 상태를 한눈에 파악"할 수 있어야 한다는 목적은 반드시 만족해야 한다.

## 메뉴 목록

| 번호(예시) | 메뉴 | 이동 대상 |
|---|---|---|
| 1 | 시료관리 | [`02-시료관리.md`](02-시료관리.md) |
| 2 | 주문(접수/승인/거절) | [`03-주문관리.md`](03-주문관리.md) |
| 3 | 모니터링 | [`04-모니터링.md`](04-모니터링.md) |
| 4 | 출고처리 | [`05-출고처리.md`](05-출고처리.md) |
| 5 | 생산라인 | [`06-생산라인.md`](06-생산라인.md) |
| 0 | 종료 | 애플리케이션 종료 |

메뉴 번호/문구는 구현 시 확정한다. 잘못된 번호 입력 시 에러 메시지를 표시하고 메인 메뉴를 다시 보여준다.

## 예외/유효성 처리

- 존재하지 않는 메뉴 번호 입력 → 안내 메시지 후 재입력 요구
- 각 하위 메뉴에서 복귀(뒤로가기) 시 항상 메인 메뉴로 돌아온다.
